| Record    | Meaning                               | What it is used for                                                                  |
| --------- | ------------------------------------- | ------------------------------------------------------------------------------------ |
| **A**     | Address record                        | Maps a domain to an IPv4 address                                                     |
| **AAAA**  | IPv6 address record                   | Maps a domain to an IPv6 address                                                     |
| **CNAME** | Canonical name                        | Makes one domain an alias of another domain                                          |
| **MX**    | Mail exchange                         | Shows which mail servers receive email for the domain                                |
| **NS**    | Name server                           | Shows which DNS servers are responsible for the domain                               |
| **SOA**   | Start of authority                    | Stores administrative information about the DNS zone                                 |
| **TXT**   | Text record                           | Stores text data, often for verification and email security                          |
| **PTR**   | Pointer record                        | Reverse DNS: maps an IP address back to a domain name                                |
| **SRV**   | Service record                        | Shows the host and port for a specific service                                       |
| **CAA**   | Certification Authority Authorization | Controls which certificate authorities may issue SSL/TLS certificates for the domain |
